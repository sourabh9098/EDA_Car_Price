# Car Price Prediction - Machine Learning Project

This project focuses on building a machine learning model to predict car prices based on various features such as engine specifications, size, and fuel type.

## Project Objective

The goal is to analyze the dataset and build a regression model that can accurately predict the price of a car.

## Dataset Features

The dataset includes:

- Car specifications (engine size, horsepower, fuel system)
- Physical features (length, width, height, weight)
- Categorical attributes (fuel type, car body, drive type)

## Data Preprocessing

- Removed unnecessary columns
- Converted categorical variables using encoding techniques
- Prepared data for machine learning models

## Machine Learning Model

- Applied regression algorithms such as:
  - Linear Regression
    
## Key Insights

- Engine size and horsepower have a strong impact on car price
- Vehicle size and weight influence pricing
- Fuel type and drive type also affect price variation

## Tools Used

- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Matplotlib  
- Seaborn  

## Conclusion

This project demonstrates how machine learning can be used to predict car prices based on various features, helping in better pricing strategies and decision making.
